# Личный план

Устанавливаемый офлайн-трекер задач, проектов и тренировок для iPhone. Все личные данные хранятся только на устройстве.

## Готовая версия

**https://drawmoney123.github.io/solo-system/**

На iPhone откройте адрес в Safari → «Поделиться» → «На экран Домой». Если на экране уже есть старая версия, откройте приложение один раз с интернетом, экспортируйте резервную копию, удалите старую иконку и добавьте сайт на экран «Домой» заново — так точно обновятся название и иконка.

## Что есть в приложении

- персональный профиль Артёма: форма, здоровье, обучение, работа, дом и семья;
- ровно 7 чётких ежедневных задач со строгой серией выполнения;
- 45 готовых разовых задач по семи категориям;
- фильтры задач по категории;
- проекты с последовательными и понятными этапами;
- отдельная система тренировок: экран «Сегодня», редактируемая неделя, Full Body A/B с гантелями дома, бег с нулевого уровня, ходьба и журнал результатов;
- самостоятельные разделы «Йога» и «Медитация» с практиками новичка, таймером, паузой, ручной записью и расписанием;
- связь тренировок и практик с ежедневным планом без двойного начисления баллов;
- персональный выбор упражнений Full Body без изменения уже сохранённой истории;
- запись подходов, повторов, веса, времени, дистанции, темпа и шагов;
- монохромный чёрно-белый интерфейс без цветовых игровых акцентов;
- баллы за выполненные дела и магазин реальных наград;
- 22 награды разной стоимости — от короткого отдыха до полезных покупок и семейной поездки;
- 9 многоэтапных проектов: физическая форма, режим, слаботочные системы, системное администрирование, безопасная теория электрики и рост дохода;
- дополнительная короткая задача на каждый день;
- статистика, график активности и измеримые вехи;
- резервная копия, восстановление с откатом и импорт пакетов;
- переключаемый архив прежнего каталога после персонализации;
- полный офлайн-режим после первого открытия.

В интерфейсе нет игровых персонажей, наказаний и специальных противников. При пропуске ежедневного плана только начинается заново серия; накопленные данные и баллы не теряются.

## Файлы

- `index.html` — интерфейс и логика приложения;
- `content.json` — основной каталог задач, проектов и наград;
- `packs/example-pack.json` — пример пакета для ручного импорта;
- `sw.js` — офлайн-кэш;
- `manifest.webmanifest`, `icon-*.png` — установка на экран «Домой».

## Запуск на компьютере

```powershell
python -m http.server 8123 --directory solo-system
# открыть http://localhost:8123
```

## Обновление контента

У каждой записи в `content.json` должен быть уникальный и постоянный `id`. При запуске с интернетом приложение добавляет записи с новыми идентификаторами и не создаёт их повторно. Пользовательские изменения и удалённые импортированные записи не перезаписываются.

Пакет можно импортировать вручную: «Ещё» → «Импорт пакета задач с ПК». Полная резервная копия создаётся через «Ещё» → «Экспорт бэкапа»; перед восстановлением приложение сохраняет текущее состояние для отката.

> JSON-копия содержит личные задачи открытым текстом. Храните её как личный файл и не импортируйте чужие копии.

## Совместимость обновления

Версия 7 сохраняет прежний ключ локального хранилища, поэтому баллы, серия, статистика, история и журнал тренировок не обнуляются. Прежние задачи, проекты и награды автоматически переносятся в переключаемый архив, а старые бэкапы остаются совместимыми. Версия офлайн-кэша поднята до `v12`.
