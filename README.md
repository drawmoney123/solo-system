# СИСТЕМА — Solo Level Up

Личное офлайн-приложение «прокачки жизни» в стиле Solo Leveling (PWA для iPhone).

## Готовая версия

**https://drawmoney123.github.io/solo-system/**

На iPhone откройте этот адрес в Safari → «Поделиться» → «На экран Домой». Если раньше была установлена версия с другого адреса, сначала экспортируйте из неё бэкап, затем установите эту версию и восстановите бэкап через вкладку «Ещё».

## Файлы

- `index.html` — всё приложение (интерфейс + логика), данные хранятся на устройстве в localStorage
- `sw.js` — service worker: полный офлайн-режим после первого открытия
- `manifest.webmanifest`, `icon-*.png` — установка на экран «Домой» iPhone
- `content.json` — контент, который приложение подтягивает с сервера при наличии сети
- `packs/example-pack.json` — пример пакета задач для ручного импорта с ПК

## Запуск локально (на ПК)

```powershell
python -m http.server 8123 --directory solo-system
# открыть http://localhost:8123
```

## Установка на iPhone

PWA требует HTTPS, поэтому папку нужно один раз выложить на бесплатный хостинг:

1. **Netlify Drop** (проще всего): открыть https://app.netlify.com/drop и перетащить папку `solo-system` — получите адрес вида `https://имя.netlify.app`.
   Либо **GitHub Pages**: залить папку в репозиторий и включить Pages.
2. На iPhone открыть адрес в **Safari**.
3. Поделиться → **«На экран "Домой"»**.
4. Готово: приложение открывается без браузерной строки, работает полностью офлайн.

## Синхронизация с компьютера (3 способа)

1. **content.json (автоматически).** Добавьте новые квесты/врата/награды в `content.json` (у каждой записи уникальный `id`), перезалейте сайт. При запуске с интернетом телефон сам подтянет новое; уже загруженные `id` не дублируются. Офлайн — просто работает дальше.
2. **Пакет задач (вручную).** Подготовьте JSON как `packs/example-pack.json`, перекиньте на телефон (Telegram «Избранное», почта, iCloud Drive) и импортируйте: «Ещё» → «Импорт пакета задач с ПК».
3. **Бэкап.** «Ещё» → «Экспорт бэкапа» сохраняет весь прогресс в JSON; «Восстановить из бэкапа» проверяет файл и загружает его обратно. Перед заменой приложение автоматически сохраняет текущее состояние, поэтому доступен откат последнего восстановления.

> Прогресс не отправляется на сервер и привязан к адресу установленного приложения. Перед сменой адреса, удалением PWA или очисткой данных Safari обязательно экспортируйте бэкап. JSON-бэкап содержит личные задачи открытым текстом — храните его как личный файл и не импортируйте чужие бэкапы.

## Механики

- **Ежедневные квесты** — привычки; сброс в полночь, за полный день +40 XP и рост серии, пропуск обнуляет серию (как штраф Системы).
- **Квесты** — разовые задачи с рангом сложности E–S (опыт и золото растут с рангом).
- **Врата (данжи)** — большие проекты, разбитые на этажи; зачищаются по порядку, последний этаж — босс.
- **Характеристики** — СИЛА (тело), ИНТЕЛЛЕКТ (учёба), КАРЬЕРА (работа), ВЫНОСЛИВОСТЬ (здоровье), ХАРИЗМА (общение), ВОЛЯ (дисциплина), ФИНАНСЫ; растут от выполненных задач соответствующего типа.
- **Фокус охотника** — до трёх приоритетных направлений (по умолчанию: СИЛА, ИНТЕЛЛЕКТ, КАРЬЕРА). Задачи фокусных направлений дают ×2 опыта и очков характеристик и помечаются бейджем ×2; настраивается на вкладке «Статус».
- **Ранг охотника** E→S — от уровня (10 / 20 / 30 / 45 / 60).
- **Лавка** — золото тратится на реальные награды, которые вы сами себе назначаете.
- **Титулы** — 20 ачивок за достижения («Пробуждение», «Железная воля», «Монарх Теней»…); выбранный титул носится под именем.
- **Скрытый квест дня** — каждый день случайное бонус-задание с двойной наградой.
- **Зона наказания** — провалил ежедневные квесты: на утро появляется штрафной квест, до его выполнения весь опыт ×½.
- **День отдыха** — награда в лавке, которая защищает текущий день: незавершённые ежедневные квесты не сбросят серию и не создадут штраф. Серия при этом не увеличивается.
- **Босс недели** — одна главная задача недели с наградой двойного A-ранга (+500 XP); живёт до понедельника.
- **Активность за 7 дней** — график выполненных дел на вкладке «Статус».
- Задачи можно редактировать — нажмите на текст задачи.

## Что исправлено в версии 3

- корректно учитываются несколько пропущенных дней и переход через полночь при открытом приложении;
- карточки и сообщения показывают фактический XP с учётом зоны наказания;
- восстановление бэкапа использует строгую проверку, ограничения размера и страховочную копию для отката;
- награда «День без квестов» действительно защищает серию;
- импорт контента ограничен и принимает только записи с уникальным `id`;
- service worker изолирует собственный кэш, не сохраняет ответы 404/500 и надёжно обновляет офлайн-оболочку;
- улучшены доступность кнопок, масштабирование и режим уменьшенной анимации.
